# Mommy & Me Collective — AI Agent Strategy App

## Overview
A Flask web application serving two HTML strategy/architecture documents for the Mommy & Me Collective AI Agent project.

## Structure
- `app.py` — Flask server serving static HTML pages on port 5000
- `steph-ai-plan.html` — AI Agent Strategy Plan document
- `steph-architecture.html` — AI Agent Architecture Map document
- `zipFile.zip` — Original uploaded zip containing both HTML files

## Routes
- `/` or `/plan` — AI Agent Strategy Plan
- `/architecture` — Architecture Map

## Tech Stack
- Python 3.11
- Flask

## Running
```
python app.py
```
Server runs on port 5000.
